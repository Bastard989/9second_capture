# 9second_capture: установка и первый запуск

**Что это за файл:** Пошаговая инструкция по установке, первому запуску и базовой настройке агента.

## Оглавление
1. Что нужно заранее
2. Установка из репозитория
3. Первый запуск
4. Быстрый запуск без мастера
5. Как выбрать провайдеры
6. Рекомендуемый безопасный подход
7. Что проверить после установки
## Что нужно заранее

Минимально подготовьте:

- `git`;
- `Python 3.10+`;
- `ffmpeg`.

Если планируется локальный LLM или локальные embeddings, понадобится локальный совместимый провайдер. Если планируются облачные провайдеры, заранее подготовьте адрес API, ключи и названия моделей.

## Установка из репозитория

1. Склонируйте проект:

```bash
git clone <repository-url>
```

2. Перейдите в папку проекта:

```bash
cd <project-directory>
```

3. Создайте виртуальное окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

4. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Первый запуск

Рекомендуемый вариант — через мастер установки:

```bash
python3 scripts/launcher.py
```

Мастер установки помогает:

- проверить окружение;
- выбрать LLM, embeddings и STT провайдеры;
- сохранить настройки без ручного редактирования конфигурации;
- открыть основной UI агента.

## Быстрый запуск без мастера

Если мастер не нужен, агент можно запустить напрямую:

```bash
python3 scripts/run_local_agent.py
```

## Как выбрать провайдеры

### LLM

Для LLM нужно указать:

- провайдера;
- API URL, если требуется;
- API key, если требуется;
- chat-модель.

### Embeddings

Для embeddings нужно указать:

- провайдера;
- embeddings API URL, если требуется;
- embeddings-модель.

### STT

Для STT нужно указать:

- провайдера;
- модель распознавания;
- секреты и endpoint, если используется облачный провайдер.

## Рекомендуемый безопасный подход

- не храните реальные ключи в Git;
- используйте только рабочие chat-модели для LLM и отдельные embeddings-модели для поиска;
- проверяйте подключение к STT через кнопку проверки, прежде чем запускать длинную транскрибацию.

## Что проверить после установки

1. UI открывается без ошибок.
2. Настройки сохраняются.
3. STT-провайдер проходит проверку подключения.
4. LLM-провайдер отвечает.
5. Embeddings-провайдер выбран отдельно от chat-модели.
