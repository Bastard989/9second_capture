# 9second_capture: состав релизного пакета

**Что это за файл:** Краткое описание состава релизного пакета и того, какие файлы передавать пользователям.

## Оглавление
1. Что входит в пакет
2. Что важно по платформам
3. Что передавать пользователям
## Что входит в пакет

В релизный пакет подготовлены:

- отдельный ZIP-архив для `macOS`;
- отдельный ZIP-архив для `Linux`;
- отдельный ZIP-архив для `Windows`;
- комплект Word-документов для публикации и передачи пользователям;
- Markdown-версии тех же документов для удобного редактирования.

## Что важно по платформам

### macOS

Пакет содержит `.app`-bundle в ZIP-архиве.

### Linux

Пакет содержит `x86_64` Linux bundle в ZIP-архиве.

### Windows

Пакет содержит Windows ZIP-архив с результатом сборки из Windows CI pipeline.

## Что передавать пользователям

Пользователям обычно достаточно:

- соответствующего ZIP-архива под их платформу;
- инструкции по установке;
- краткого руководства пользователя;
- файла с типовыми ошибками.
