# QA Checklist (Local Agent)

Актуальный чеклист под текущий UI и поток работы агента.

## 1. Старт и доступность

1. Запустить агент:

```bash
cd "/Users/kirill/Documents/New project/9second_capture"
python3 scripts/run_local_agent.py
```

2. Проверить, что UI открывается и не падает на старте.
3. Проверить, что переключение RU/EN работает.

## 2. Режимы

1. Переключить режимы:
- `Браузер: экран + звук`
- `API/файл: загрузка аудио`
- `Ссылка: quick fallback`
2. Убедиться, что видны корректные подсказки и настройки каждого режима.

## 3. Запись

1. Нажать `Старт` в браузерном режиме.
2. В окне браузера выбрать вкладку/экран и включить `Share audio`.
3. Убедиться, что статус записи меняется на активный.
4. Нажать `Стоп`.
5. Проверить, что появляется запись в списке `Результаты -> Аудио`.

## 4. MP3 операции

1. Выбрать запись в `recordsSelect`.
2. Нажать `Сохранить MP3` и проверить сохранение файла.
3. Нажать `Импорт MP3`, выбрать внешний файл и проверить, что запись добавлена в список.
4. Проверить, что импорт MP3 не запускает STT автоматически.

## 5. Транскрипция (ручной запуск)

1. Вкладка `Результаты -> Транскрипция`.
2. Для `Raw` нажать `TXT`, проверить, что создаётся `raw`-файл.
3. Для `Clean` нажать `TXT`, проверить, что создаётся `clean`-файл.
4. Проверить, что до нажатия кнопок транскрипт не строится.

## 6. LLM по файлам (ручной запуск)

1. Вкладка `Результаты -> LLM`.
2. Прикрепить TXT/CSV/JSON/MD.
3. Выполнить запрос на таблицу и JSON.
4. Проверить, что создаются файлы результатов.
5. Нажать `Очистить` и проверить очистку истории/предыдущих результатов.

## 7. Настройки моделей

1. `STT модель`: `Сканировать` -> `Сменить модель`.
2. `LLM модель`: `Сканировать` -> выбрать chat-модель -> `Сменить модель`.
3. `Embeddings модель`: проверить отдельный список embedding-моделей.
4. Проверить, что embedding-модель не попадает в выбор chat-LLM.

## 8. Busy overlay

1. Запустить длительную операцию (STT/LLM).
2. Проверить отображение прогресса в процентах.
3. Проверить `Свернуть` и возврат из мини-окна.
4. Проверить `Прервать` для поддерживаемых операций.
