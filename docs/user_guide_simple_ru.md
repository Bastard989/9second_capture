# 9second_capture: простой гайд

Агент записывает встречу в MP3, затем по вашему запросу делает транскрипт и LLM-форматы.

## Что важно

- Запись/импорт MP3 не запускают транскрипт автоматически.
- Транскрипт запускается только вручную во вкладке `Транскрипция`.
- LLM запускается только вручную во вкладке `LLM`.

## Быстрый сценарий

1. Выберите режим (`Браузер`, `API/файл`, `Ссылка`).
2. Нажмите `Старт`, затем `Стоп`.
3. Во вкладке `Аудио` сохраните MP3 или импортируйте внешний MP3.
4. Во вкладке `Транскрипция` нажмите `TXT` для `Raw` или `Clean`.
5. Во вкладке `LLM` прикрепите файлы и отправьте запрос на нужный формат.

## Что можно получить из LLM

- таблицы для Google Sheets,
- JSON-структуры,
- CSV,
- текстовые summary и сводки.

## Если медленно

- Для STT выберите модель `small`.
- Для качества выше используйте `medium`/`large-v3`.
- Для LLM убедитесь, что выбрана chat-модель, а не embeddings-модель.

## Запуск

```bash
cd <project-directory>
python3 scripts/run_local_agent.py
```
