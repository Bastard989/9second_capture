# 9second_capture: простой гайд (без сложных терминов)

Этот агент помогает записать встречу, сделать из звука текст и подготовить отчеты для команды.

## Что у вас получится

- сохранить запись встречи в `MP3`;
- сделать два вида текста:
  - `Raw` (грязный, как услышали);
  - `Clean` (чистый, без лишних "эээ", "ммм", "ну");
- сохранить отчеты на компьютер;
- конвертировать отчеты в `TXT`, `JSON`, `CSV`, `Table JSON`;
- сделать отдельный `Senior brief` (краткий отчет для руководителя/сеньора).

## Самый важный сценарий (после встречи)

1. Нажмите `Стоп`, когда встреча закончилась.  
2. Агент спросит имя MP3 файла. Введите имя и сохраните MP3 в нужную папку.  
3. В блоке `Результаты`:
   - в секции `1) MP3 после встречи` выберите запись;
   - если нужно, нажмите `Сохранить MP3` еще раз;
   - если MP3 уже у вас на компьютере, нажмите `Импорт MP3`.
4. В секции `2) TXT отчет из MP3`:
   - выберите `Raw` или `Clean`;
   - впишите имя файла отчета;
   - нажмите `Сформировать TXT`.
5. В секции `3) Конвертация готовых отчетов`:
   - в левой плашке работайте с `Raw` отчетом;
   - в правой плашке работайте с `Clean` отчетом;
   - выбирайте нужный формат: `TXT`, `JSON`, `CSV`, `Таблица JSON`, `Для сеньоров`.

## 4 режима работы (какой когда выбрать)

- `Драйвер: системный звук`  
  Обычная запись звонка/встречи с системного звука.
- `Браузер: экран + звук`  
  Захват через браузер. Важно включить `Share audio`.
- `API/файл: загрузка аудио`  
  Когда файл встречи уже есть, и вы хотите обработать его как пост-фактум.
- `Ссылка: quick fallback`  
  Резервный режим, если основной способ захвата недоступен.

## Куда сохраняются файлы

- Вы выбираете папку сами:
  - через кнопку `Выбрать папку`, или
  - через системное окно сохранения файла.
- Агент не отправляет данные в облако автоматически: основной сценарий локальный.

## Если что-то не работает

- Нет звука в записи: проверьте источник аудио и разрешения браузера.
- В режиме браузера нет звука собеседника: включите `Share audio`.
- Нет текста после записи: сохраните MP3 и попробуйте снова через `Импорт MP3`.
- Не создается отчет: сначала выберите запись, затем `Raw/Clean`, затем имя файла.
